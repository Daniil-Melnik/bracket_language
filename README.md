<h1>Программа "Синтаксический анализатор скобочной грамматики"</h1>
<p>Реализует автомат по проверке скобчного выражения на соответсвие установленным правилам</p>
<h2>Задание</h2>
<p>Правильная скобочная запись арифметических выражений с тремя видами скобок. Внутри фигурных скобок обязательно должны быть квадратные, но могут быть круглые, внутри квадратных должны быть круглые или бесскобочные выражения, внутри круглых только бесскобочные арифметические выражения. Также, если один из операндов является скобкой, то и второй должен быть скобкой (не может быть буквой). Могут быть лишние скобки, но одна буква не может браться в скобки.</p>
